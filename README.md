# Demo Penanganan Peristiwa

## Deskripsi
Proyek ini merupakan implementasi dasar penanganan peristiwa (event handling) menggunakan JavaScript pada halaman web. Setiap interaksi pengguna seperti klik, input, pemilihan, dan pergerakan mouse diproses melalui event listener untuk menghasilkan respons yang terlihat pada area output.

## Tujuan
- Memahami konsep event pada JavaScript
- Menerapkan berbagai jenis event pada elemen HTML
- Menghubungkan interaksi pengguna dengan perubahan tampilan secara dinamis

## Fitur
Aplikasi ini mencakup beberapa jenis event berikut:

- **Click Event**  
  Menampilkan pesan ketika tombol diklik

- **Input Event**  
  Menampilkan teks secara langsung saat pengguna mengetik

- **Change Event**  
  Menampilkan pilihan pengguna dari dropdown

- **Submit Event**  
  Memproses form tanpa melakukan reload halaman

- **Keyup Event**  
  Menampilkan tombol terakhir yang ditekan

- **Mouseover & Mouseout Event**  
  Mengubah tampilan saat mouse masuk dan keluar dari area tertentu

- **Focus & Blur Event**  
  Memberikan efek visual saat input aktif dan tidak aktif
